# Expense Tracker React App

> npm install   
> cd client   
> npm install   
> cd ..   

### Run front and backend both
> npm run dev   

### Run Backend only
> npm run server   

### Run Frontend only
> npm run client   

### Build client
> cd client   
> npm run build   

### Run Prod
> cd ..  
> npm start
